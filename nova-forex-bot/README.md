# Nova HFT Bot: Institutional Trading Engine

A state-of-the-art, professional-grade High-Frequency Trading (HFT) system designed for the MetaTrader 5 (MT5) platform. Built with a dual-model ensemble brain and volatility-dynamic risk management.

## 🚀 Core Features

- **Dual-Model Ensemble**: Combines XGBoost and Random Forest for high-confidence signal generation.
- **Tick Data Engine**: Real-time micro-structure feature engineering (Velocity, Momentum, Spread Imbalance).
- **Bidirectional Execution**: Intelligent Buy/Sell logic with dynamic Kelly Criterion lot sizing.
- **Institutional Dashboard**: A premium Streamlit interface for live monitoring and risk diagnostics.
- **Robust Safety**: Advanced Kill-Switches for spread spikes, latency surges, and circuit breakers for loss streaks.

## 🛠️ Technology Stack

- **Language**: Python 3.10+
- **Trading Plateform**: MetaTrader 5
- **AI/ML**: XGBoost, Scikit-Learn, Pandas
- **Frontend**: Streamlit, Plotly
- **Communication**: WebSockets, Telegram API
